# Гарант Бани — сайт

Репозиторий разработки сайта «Гарант Бани» в визуальной концепции **«Северный пар»**.

## Принцип разработки

- работа ведётся короткими тематическими Pull Request;
- `main` хранит только согласованные решения;
- дизайн в Figma и реализация в коде должны использовать единую систему компонентов и ассетов;
- реальные фотографии продукции не подменяются вымышленными характеристиками;
- временные AI-концепты явно маркируются как concept / placeholder.
