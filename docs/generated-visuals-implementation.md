# Временные визуалы в рабочем сайте

## Статус

В ветке `feat/site-with-generated-visuals` собран первый рабочий статический лендинг на основе утверждённой Figma-концепции.

Сгенерированные изображения встроены в проект и загружаются локально из репозитория без внешнего фотохостинга.

## Карта использования

| Ключ | Использование |
|---|---|
| `hero` | первый экран, главный кейс, финальный CTA |
| `barrel` | баня-бочка, квадро-овал, дополнительная работа |
| `trailer` | баня на прицепе, альтернативный кадр проекта |
| `interior` | интерьер парной / моечной |

Изображения являются временными AI-концептами. Они задают требования к будущей фотосъёмке, но не должны публиковаться как подтверждённые работы компании.

Запись изображений в Figma временно остановлена лимитом Figma MCP на Starter-плане. После восстановления лимита используются те же подготовленные ассеты без повторной генерации.

## Следующий этап

1. Подтвердить структуру лендинга и визуальную иерархию.
2. Включить GitHub Pages или перенести статическую версию в выбранный стек.
3. Подключить реальные формы заявок.
4. Заменить AI-концепты проверенными фотографиями объектов.
5. Подготовить WebP/AVIF в production-разрешениях.
