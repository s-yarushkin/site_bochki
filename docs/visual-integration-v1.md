## Правила замены реальными фото

1. Не менять утверждённую композиционную роль кадра.
2. Проверять, что объект действительно произведён «Гарант Бани».
3. Не переносить на фото вымышленные окна, террасы, печи или комплектации.
4. Для Hero сохранять объект справа и чистую зону слева.
5. Для каталога каждый формат должен считываться по силуэту без подписи.
6. Подготовить отдельные desktop и mobile crops, а не полагаться только на `object-position`.

## Следующий шаг

После получения оригинальных фотографий из VK/архива:

- отобрать 3–5 объектов каждого формата;
- провести конструктивную сверку;
- выполнить единую цветокоррекцию;
- заменить concept-сцены в Figma;
- экспортировать AVIF/WebP для реализации сайта.
