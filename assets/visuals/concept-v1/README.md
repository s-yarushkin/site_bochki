# Concept Visuals V1

> Все изображения в этой папке — AI-концепты для проверки композиции и арт-дирекции. Они не являются фотографиями реальных объектов «Гарант Бани» и должны быть заменены подтверждёнными материалами перед production-релизом.

## Файлы

- `hero-north-steam-concept.{jpg,webp}` — desktop hero, 1600×900, затемнение слева под оффер.
- `hero-north-steam-concept-mobile.webp` — mobile hero, 768×1120.
- `format-barrel-concept.*` — баня-бочка.
- `format-quadro-concept.*` — квадро-овал.
- `format-frame-concept.*` — каркасная баня.
- `format-trailer-concept.*` — баня на прицепе.
- `*-mobile.webp` — mobile crops.
- `product-frame-concept.*` — квадратный product/case asset.
- `visual-concepts-board-v1.jpg` — обзорная доска.

## Правила использования

1. Не публиковать как «наши работы».
2. Не связывать с ценой, размером или комплектацией.
3. Использовать только с маркировкой `concept` до замены реальными фото.
4. При замене сохранять композицию, crop и световой баланс.
